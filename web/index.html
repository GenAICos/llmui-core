📋 Fonctionnalités LLMUI Core

Interface de consensus multi-modèles

  • Mode Simple : Conversation directe avec un modèle unique
  • Mode Consensus : Plusieurs modèles analysent votre requête
  • Multi-fichiers : Support de plusieurs fichiers simultanément
  • Tri alphabétique : Modèles organisés pour faciliter la sélection
  • Édition en ligne : Modifiez votre dernier prompt directement
  • Thème clair/sombre : Interface adaptative
  • Mémoire de session : Conservation du contexte conversationnel
  • 🔐 Authentification : Connexion sécurisée et gestion des sessions

Formats supportés :

  • Fichiers texte (.txt, .md)
  • Code source (.py, .js, .json, .sh, .css, .html, .xml, .yaml, .yml, .sh)
  • Données (.csv)
  • Bureautique (.docx, .xlsx, .pdf)
